## 💡 게임 개발 팁

### Teachable Machine 모델 학습 시 주의사항

1. **다양한 환경에서 샘플 수집**
   - 밝은 곳 / 어두운 곳
   - 다양한 배경
   - 여러 각도와 거리

2. **충분한 샘플 수집**
   - 각 클래스당 최소 50개 이상 권장
   - 더 많을수록 정확도 향상

3. **대기 상태 클래스 추가**
   - "정면" 또는 "대기" 클래스를 만들어 아무 포즈도 취하지 않은 상태를 학습
   - 오인식 방지에 도움

### gameEngine.js 커스터마이징 가이드

게임 로직을 구현할 때 다음 사항을 참고하세요:

```javascript
// js/main.js에서 게임 모드 활성화
startGameMode({
  timeLimit: 60,  // 여기서 정의한 제한 시간
  commands: ["왼쪽", "오른쪽", "위", "아래"]  // 여기서 정의한 포즈 목록
});
```
